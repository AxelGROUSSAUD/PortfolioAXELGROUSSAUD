J'ai eu l'occasion de réaliser un site statique sur Picasso dans le cadre des projets personnels encadrés (PPE) du BTS SIO option SLAM en première année au Lycée Saint Michel à Annecy. Pour réaliser ce site j'ai utilisé les languages HTML, CSS et Javascript avec la bibliothèque JQuerie. Le Javascript servait essentiellement à afficher des images aléatoire dans un carrousel. Ensuite j'ai fait évoluer le site en le rendant dynamique avec PHP.
